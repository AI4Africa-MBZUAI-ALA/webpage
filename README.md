# AI for Africa Bootcamp 2026 Website

Static documentation site for the official AI for Africa Bootcamp 2026 website.

## Local development

1. Create an isolated environment with `uv venv .venv`.
2. Install dependencies with `uv pip install --python .venv/bin/python -r requirements.txt`.
3. Run `.venv/bin/mkdocs serve` to preview the site locally.

## Build

Run `.venv/bin/mkdocs build --strict` to generate the static site in `site/`.
