# Program Overview

The AI for Africa Bootcamp 2026 is built as a two-week residential experience that combines AI learning, leadership development, and project work.

## Core elements

- AI foundations and applied understanding
- Responsible use of AI and ethics
- Leadership, teamwork, and communication
- Community challenge framing and solution design
- Mentored project development and presentation

## Learning approach

The program is documentation-first, workshop-friendly, and designed to be easy for mentors to support. The site can be updated as the schedule or learning modules evolve.
